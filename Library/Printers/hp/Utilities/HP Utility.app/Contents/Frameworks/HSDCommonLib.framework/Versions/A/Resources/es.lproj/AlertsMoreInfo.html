Mostrar el contenido web en las alertas

Para mejorar su experiencia, HP puede comprobar si está inscrito en un programa HP o si es elegible para ofertas especiales, y modificar la alerta según las opciones disponibles.

La siguiente información puede ser enviada de forma segura a HP:
  • Información sobre el dispositivo como modelo y número de serie
  • Información sobre el sistema operativo como versión, idioma y país
  • Información sobre la alerta
  • Un identificador de usuario asignado por HP
