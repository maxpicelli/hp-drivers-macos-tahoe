Plus d'informations

La participation à ce programme de recherche exige un accès Internet et votre accord pour la collecte par HP et la transmission à HP de vos données d'utilisation (ces données sont présentées ci-dessous).

Pedant la connexion HP, un partage périodique de ces données se produira automatiquement.

Données collectées par HP :

  • Informations de base sur votre Mac : système d'exploitation, langue, fuseau horaire.
  • Identification de base de l’imprimante : modèle, numéro de série.
  • Volumes d'impression : Nombre/type de cartouches, types de pages imprimées.
  • Événements d'impression : avertissements sur niveau bas des cartouches.
  • Fonctionnalités de l’imprimante que vous utilisez : Cartes avec photos, télécopies, scanner, Serveur Web Incorporé.
  • Logiciels utilisés pour imprimer : nom de chaque application et nombre de pages imprimées avec chaque application.
