Das Fenster "Digitales Faxarchiv" dient zum Einrichten des HP Geräts, um empfangene Schwarzweißfaxe direkt an einen Ordner auf einen Hostcomputer (lokal oder im Netzwerk) oder an eine angegebene E-Mail-Adresse zu senden. Die Archivierung für die E-Mail-Funktion wird nicht von allen HP Geräten unterstützt.

Die digitale Faxarchivierung ist nur für Schwarzweißfaxe verfügbar. Farbfaxe werden normalerweise bei deren Empfang immer gedruckt.

Wählen Sie die Schaltfläche aus, die der gewünschten Archivierung von Faxen entspricht: "In Ordner speichern" oder "An E-Mail weiterleiten".

Klicken Sie zum Speichern in einem Ordner auf "Ordner auswählen…", um den gemeinsam genutzten Ordner auszuwählen, der zum Speichern empfangener Faxe verwendet werden soll. Geben Sie die Daten direkt in die Felder ein, oder navigieren Sie zum gewünschten Ordner. Je nach Ihrem speziellen Gerät oder dem ausgewählten Ordner müssen Sie die entsprechenden Server- oder Authentifizierungsdaten eingeben.

Klicken Sie auf die Schaltfläche "Verbinden", um den Verknüpfungspunkt auf Ihrem Computer einzuhängen, über den Sie auf Wunsch zu einem Unterordner navigieren können.

Geben Sie für "An E-Mail weiterleiten" den Empfänger und die Daten für den Server (SMTP) Server für ausgehende E-Mails ein, der von Ihrem HP Gerät verwendet werden soll. Fragen Sie bei Bedarf Ihren Netzwerkadministrator oder Ihren Internet Service Provider (ISP) nach diesen Daten.

Damit Sie beim Empfang von Faxen auf Ihrem Computer informiert werden, klicken Sie auf die Schaltfläche "Benachrichtigung über eingehende Faxe: Ein".

WICHTIG! Die Funktion "In Ordner speichern" erfordert einen gemeinsam genutzten Ordner. In der Dokumentation zu Ihrem Computer finden Sie die Informationen zum Einrichten eines gemeinsam genutzten Ordners.

HINWEIS: Über den integrierten Webserver auf Ihrem HP Gerät können Sie auf weitere Informationen und erweiterte Einstellungen zugreifen. Weitere Informationen finden Sie in der Dokumentation zum Gerät.
