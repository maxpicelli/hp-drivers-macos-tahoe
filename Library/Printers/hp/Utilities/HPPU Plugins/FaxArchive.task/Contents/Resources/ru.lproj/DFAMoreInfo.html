Панель "Архив цифровых факсов" позволяет настроить устройство HP для отправки полученных черно-белых факсов непосредственно в папку на локальном или сетевом компьютере или на указанный адрес электронной почты. Не все устройства HP поддерживают архивацию по адресу электронной почты.

"Архивация цифровых факсов" действует только для черно-белых факсов. Цветные факсы при получении будут печататься обычным образом.

Выберите кнопку, соответствующую требуемому способу архивации факсов: "Сохранить в папке" или "Переслать по эл. почте".

При выборе "Сохранить в папке" щелкните "Выбрать папку..." для выбора общей папки, в которой требуется сохранять полученные факсы. Введите имя непосредственно в поле либо перейдите к требуемой папке. В зависимости от конкретного устройства и выбранной папки потребуется ввести соответствующую информацию для сервера и проверки подлинности.

Нажмите кнопку "Подключить", чтобы подключить общую точку на локальном компьютере. Это позволит перейти в подпапку, если это потребуется.

При выборе параметра "Переслать по эл. почте" введите получателя и информацию о сервере исходящей почты (SMTP), который будет использоваться устройством HP. Если эти данные неизвестны, обратитесь к администратору сети или поставщику услуг Интернета (ISP).

Чтобы получать уведомления при получении факса на компьютере, нажмите кнопку "Уведомление о входящем факсе: Вкл.".

ВНИМАНИЕ! Для сохранения в папке требуется общая папка. Информацию о создании общей папки см. в документации компьютера.

ПРИМЕЧАНИЕ. Для настройки дополнительных параметров и получения доступа к другим функциям можно использовать Встроенный Web-сервер устройства HP. Дополнительные сведения см. в документации устройства.
