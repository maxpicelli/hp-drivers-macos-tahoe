O painel Arquivo de fax digitall é usado para configurar o dispositivo HP para enviar faxes em preto-e-branco recebidos diretamente a uma pasta em um computador host local ou em rede ou a um endereço de email especificado. Nem todos os dispositivos HP suportam arquivamento para email.

Arquivo de fax digital funciona apenas para faxes em preto-e-branco. Os faxes em cores serão sempre impressos normalmente quando recebidos.

Selecione o botão correspondente ao modo como gostaria de arquivar faxes: Salvar em Pasta ou Encaminhar para Email.

Para Salvar em Pasta, clique em "Selecionar Pasta…" para selecionar a pasta compartilhada que deseja usar para armazenar faxes recebidos. Digite diretamente nos campos ou navegue até a pasta que deseja usar. Dependendo do dispositivo específico e da pasta selecionada, será preciso fornecer as informações apropriadas de servidor e autenticação.

Clique no botão "Conectar" para montar o ponto de compartilhamento no computador, permitindo que você navegue até uma subpasta se desejado.

Em Encaminhar para Email, forneça o destinatário e as informações do servidor de email de saída (SMTP) que o dispositivo HP deve usar. Se não souber essas informações, verifique-as com seu administrador de rede ou com o provedor de serviços de Internet (ISP).

Para ser notificado quando um fax for recebido no computador, selecione "Notificação de Recebimento de Fax: Ativado".

IMPORTANTE! Salvar em Pasta requer uma pasta compartilhada. Consulte a documentação do seu computador para saber como configurar uma pasta compartilhada.

NOTA: É possível acessar mais opções e configurações avançadas usando o Servidor Web incorporado ao dispositivo HP. Consulte a documentação do dispositivo para mais informações.
