Le volet Archive fax numérique permet de configurer votre périphérique HP pour envoyer les télécopies noir et blanc reçues directement dans un dossier sur un ordinateur local ou sur un ordinateur hôte réseau, ou encore à une adresse électronique spécifiée Certains périphériques HP ne prennent pas en charge l'archivage vers le courrier électronique.

Archivage numérique des télécopies fonctionne uniquement pour des télécopies noir et blanc. Les télécopies couleur s'impriment toujours normalement à réception.

Sélectionnez le bouton correspondant au mode d'archivage souhaité pour les télécopies : Enregistrer dans un dossier ou Transférer à courrier électronique.

Pour Enregistrer dans un dossier, cliquez sur «Choisir un dossier…» pour sélectionner le dossier partagé que vous souhaitez utiliser pour stocker les télécopies reçues. Tapez directement dans les champs ou naviguez jusqu'au dossier que vous souhaitez utiliser. Selon votre périphérique spécifique et le dossier sélectionné, vous devrez entrer les informations de serveur et d'authentication appropriées.

Cliquez sur le bouton «Connecter» pour monter le point de partage sur l'ordinateur permettant de naviguer jusqu'à un sous-dossier si vous le souhaitez.

Pour Transférer à courrier électronique, entrez le destinataire et les informations du serveur de messagerie sortant (SMTP) que votre périphérique HP doit utiliser. Si vous ne connaissez pas ces informations, demandez-les à votre administrateur réseau ou à votre fournisseur de services Internet (ISP).

Pour être averti de la réception d'une télécopie sur votre ordinateur, sélectionnez le bouton «Notification des télécopies entrantes : Activé».

IMPORTANT! Enregistrer dans un dossier nécessite un dossier partagé. Reportez-vous à la documentation de votre ordinateur pour apprendre comment configurer un dossier partagé.

REMARQUE: vous pouvez accéder à d'autres options et à des paramètres avancés à l'aide du Serveur Web Incorporé de votre périphérique HP. Pour plus d'informations, reportez-vous à la documentation du périphérique.
