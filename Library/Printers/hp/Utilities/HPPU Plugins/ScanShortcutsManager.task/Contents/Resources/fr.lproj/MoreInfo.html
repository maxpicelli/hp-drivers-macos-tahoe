L'option Numériser vers un ordinateur vous permet de lancer des numérisations à partir de votre périphérique HP et d'envoyer automatiquement le résultat sur votre ordinateur.

Pour utiliser la fonction Numériser vers un ordinateur, assurez-vous que l'option Activer la numérisation vers PC est sélectionnée.

Lorsque HP Easy Scan (ou HP Scan pour certains produits) est installé, vous avez également la possibilité d'afficher et de modifier les tâches de numérisation. Pour afficher et modifier les tâches de numérisation, cliquez sur le bouton Édition des tâches de numérisation. Pour télécharger et installer la dernière version du logiciel pour votre appareil, rendez-vous sur http://www.hp.com/support.

Si votre périphérique HP est partagé avec d'autres personnes, vous devrez peut-être sélectionner le nom de votre ordinateur sur le périphérique lorsque vous lancez une numérisation. Il s'agit du nom d'ordinateur spécifié dans les Préférences Système de votre Mac. Si vous ne vous rappelez pas ce nom, ou si vous souhaitez vous assurer qu'il est unique et facilement reconnaissable, cliquez sur Modifier le nom de l'ordinateur.
