Digitalizar para computador permite iniciar digitalizações em seu dispositivo HP e enviar a saída automaticamente para o computador.

Para usar Digitalizar para computador, confira se Ativar Digitalizar para computador está selecionado.

Com o HP Easy Scan (ou HP Scan para alguns produtos) instalado, você pode também visualizar e editar as tarefas de digitalização. Para visualizar e editar as tarefas de digitalização, clique no botão Editar tarefas de digitalização. Para baixar e instalar o software mais recente do dispositivo, acesse http://www.hp.com/support.

Se seu dispositivo HP for compartilhado com terceiros, você pode ter que selecionar o nome de seu computador no dispositivo ao iniciar a digitalização. Esse é o nome de computador especificado nas preferências do sistema de seu Mac. Caso não se lembre do nome, ou se quiser garantir que o nome seja exclusivo e fácil de reconhecer, clique em Alterar nome do computador.
