Панель "Основные параметры факса" используется для ввода параметров устройства HP, необходимых для отправки и получения факсов.

Поля "Организация/Имя" и "Номер факса" будут печататься в заголовке каждой страницы, отправляемой по факсу. Для отправки факсов устройству требуются имя станции и номер факса отправителя. Для продолжения заполните поля "Организация/Имя" и "Номер факса".###MoreInfoFaxDialPrefix###

Звонков до ответа позволяет устройству автоматически отвечать на входящие вызовы после указанного количества звонков.

Для проверки линии факса, подключенной к устройству, нажмите кнопку "Проверка настройки факса устройства".

ПРИМЕЧАНИЕ. Дополнительные сведения см. в документации устройства.
