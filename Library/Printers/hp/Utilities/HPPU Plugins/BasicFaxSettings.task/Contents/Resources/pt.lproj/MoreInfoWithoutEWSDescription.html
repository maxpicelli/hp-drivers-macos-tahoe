O painel Configurações básicas de fax é usado para inserir as configurações de que seu dispositivo HP necessita para enviar e receber faxes.

Os campos "Empresa/Nome" e "Número do Fax" serão impressos no cabeçalho na parte superior de cada página enviada via fax. O nome da estação e o número de fax do remetente são necessários para o dispositivo enviar faxes. Para continuar, preencha os campos "Empresa/Nome" e "Número do Fax".###MoreInfoFaxDialPrefix###

Toques até atender permite ao dispositivo atender automaticamente as chamadas após o número especificado de toques.

Para testar a linha de fax conectada ao seu dispositivo, clique no botão "Testar a Configuração de Fax de Dispositivo".

NOTA: Consulte a documentação do dispositivo para mais informações.
