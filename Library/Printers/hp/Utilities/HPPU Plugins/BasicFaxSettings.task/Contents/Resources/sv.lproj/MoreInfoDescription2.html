I fönstret Faxgrundinställningar anges de inställningar som HP-enheten behöver för att skicka och ta emot fax.

Fälten "Företag/namn" och "Faxnummer" skrivs ut i sidhuvudet överst på varje sida som du faxar. För att enheten ska kunna skicka fax krävs ett stationsnamn och ett avsändarfaxnummer. För att gå vidare måste du fylla i fälten "Företag/namn" och "Faxnummer".###MoreInfoFaxDialPrefix###

Med Ringsignaler före svar kan enheten automatiskt besvara inkommande samtal efter ett specificerat antal ringsignaler.

OBS! Du kan få tillgång till fler alternativ och avancerade inställningar via HP-enhetens inbäddade webbserver. Gå till dokumentationen till enheten om du vill veta mer.
