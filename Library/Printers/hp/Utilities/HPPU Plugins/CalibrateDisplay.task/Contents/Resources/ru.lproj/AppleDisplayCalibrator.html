Инструкции по калибровке монитора с помощью приложения Apple Display Calibrator Assistant
 
Со временем качество изображения (цветопередача монитора) ухудшается. Профиль ICC, который поставляется с некоторыми мониторами, постепенно становится недействительным и точность цветопередачи ухудшается. Для создания специального профиля ICC и достижения оптимальной цветопередачи регулярно используйте приложение Apple Display Calibrator. Поскольку периодичность калибровки у разных мониторов различна, специалисты советуют выполнять калибровку монитора раз в две недели.
 
Чтобы открыть приложение Apple Display Calibrator Assistant, выполните следующие действия.
 
1. Откройте окно System Preferences (Системные параметры) и щелкните на значке Displays (Мониторы).
 
2. Щелкните на кнопке Color (Цвет).
 
3. Щелкните на кнопке Calibrate (Калибровать). С помощью приложения Display Calibrator Assistant выполните все этапы калибровки монитора. Для получения более подробных сведений о калибровке монитора отметьте флажок Expert Mode (Режим эксперта).
