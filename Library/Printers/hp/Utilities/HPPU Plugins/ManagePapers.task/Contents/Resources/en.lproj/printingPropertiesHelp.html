Choosing Printing Properties

Choosing the correct printing properties:

  • Paper Thickness: refers to carriage height or platen gap. You should choose high if your paper weights over 250gr or if it does not have plastic coating.
  • Dry time: the recommendation is to use optimized by default and move to extended if necessary.
  • Starwheels: print with the starwheels up or down.
  • Cutter: Disable or enable cutter.
  • Global Ink Volume/Gloss Enhancer Amount: initially it can be left as it is and modify it later on, based on printing results. (see point 3)
  • Vacuum level: This is relevant to improve paper advance and to avoid paper crashes. Increase the level if you have a very curled paper that is insufficiently flat on the print platen, or decrease it if your paper cannot advance due to the high suction.

When an option is grayed-out it is not available for the selected paper type.

In order to optimize printing parameters, the following steps are recommended:

  1. Choose paper type and advanced settings
  2. Perform a calibration
  3. Print an image of your choice, where you can check for color gamut, coalescence, mottle, bronzing and gloss uniformity.
  4. If the results are not satisfactory, repeat the process, paying special attention to:
    • For better color gamut increase ink volume OR select a "more ink" variant of the same paper type if available.
    • To reduce coalescence decrease ink volume and/or gloss enhancer amount OR select a "less ink" variant of the same paper if available.
    • To reduce bronzing or improve gloss uniformity adjust gloss enhancer amount.
