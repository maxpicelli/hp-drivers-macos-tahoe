Seleccionando las propiedades de impresión correctas

Seleccionando las propiedades de impresión correctas:

  • Grosor de papel: Este parámetro es importante para la altura del carro o el espacio de la platina. Debe seleccionar "Alto" si el gramaje del papel supera los 250 g/m² o si no está plastificado.
  • Tiempo de secado: lo recomendable es utilizar el valor optimizado por defecto y cambiar al valor extendido en caso necesario.
  • Ruedas de estrella: Imprimir con las ruedas de estrella levantadas o bajadas.
  • Cuchilla: Activar o desactivar la cuchilla.
  • Volumen de tinta global/Nivel de mejora de satinado: Se puede dejar inicialmente tal y como está y modificarlo posteriormente en función de los resultados de la impresión.
  • Nivel de vacío: es importante para mejorar el avance de papel y para evitar roturas de papel. Aumente el nivel si tiene papel muy curvado que no queda lo suficientemente liso en la platina de impresión, o reduzca el nivel si el papel no puede avanzar debido a una aspiración alta.
  • Calibración de avance de papel: valor de paso para definir la resolución de avance del papel.

Cuando una opción está atenuada, no está disponible para el tipo de papel seleccionado.

Para optimizar los parámetros de impresión, se recomienda efectuar los pasos siguientes:

  1. Seleccione el tipo de papel y los ajustes avanzados
  2. Realice una calibración
  3. Imprima una imagen de su elección para que pueda comprobar la gama de color, la coalescencia, el moteado, la metalización y la uniformidad del brillo.
  4. Si los resultados no son satisfactorios, repita el proceso y tenga en cuenta que:
    • Para obtener una gama de color mejorada, aumente el volumen de tinta o, si es posible, seleccione una alternativa con más tinta para el mismo tipo de papel.
    • Para reducir la coalescencia, reduzca el volumen de tinta y el nivel de mejora de satinado o, si es posible, seleccione una alternativa con menos tinta para el mismo tipo de papel.
    • Para reducir la metalización o mejorar la uniformidad del brillo, ajuste el nivel de mejora de satinado.
