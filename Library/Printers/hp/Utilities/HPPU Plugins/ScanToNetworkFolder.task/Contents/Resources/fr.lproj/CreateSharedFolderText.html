REMARQUE : les éléments numérisés à partir de votre périphérique HP peuvent uniquement être enregistrés sur un dossier partagé SMB (Windows). Suivez les étapes ci-dessous pour créer un dossier partagé compatible avec les fonctionnalités de votre périphérique HP.

1. Activer le partage de fichiers SMB :
(a) Choisissez Menu Apple > Préférences Système…, et cliquez sur Partage.
(b) Sélectionnez Partage de fichiers dans la liste, puis cliquez sur Options….
(c) Sélectionnez Partager les fichiers et dossiers via SMB (Windows).
(d) Sélectionnez la case à cocher en regard du compte d'utilisateur qui sera utilisé pour le partage.
(e) Entrez le mot de passe pour l'utilisateur sélectionné et cliquez sur OK.
(f) Cliquez sur Fermer pour poursuivre avec l'étape 2.

2. Sélectionner un dossier spécifique à partager :
(a) Cliquez sur le bouton Ajouter (+) au bas de la liste Dossiers partagés.
(b) Localisez le dossier, sélectionnez-le et cliquez sur Ajouter.
