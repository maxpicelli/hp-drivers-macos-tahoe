Paso 1 - Calibrar
La calibración de la impresora es necesaria para generar colores que puedan repetirse con una combinación de papel, tinta e impresora. Para ello, lineariza la impresora y determina la densidad máxima que se puede conseguir con cada una de las tintas. El proceso consiste en la impresión y la medición de parches de diferentes densidades con cada una de las tintas.

Paso 2 - Finalizado
Una vez finalizada la calibración, puede consultar la lista de todos los tipos de papel que reconoce su impresora. Para cada tipo de papel se indica: la categoría de papel, el nombre de papel, el estado de calibración y la fecha de calibración. Si en el estado de calibración de un determinado papel aparece "OK" (correcto), significa que dicho papel ha sido calibrado recientemente y está listo para ser utilizado. Si el estado es "Obsolete" (obsoleto), significa que se ha realizado una calibración al menos una vez, pero se considera que ha transcurrido demasiado tiempo desde entonces para que ésta resulte fiable, o ya no se considera válida, por ejemplo, debido al cambio de un cabezal de impresión. Deberá volver a calibrar el papel para garantizar los mejores resultados. Si el estado es "Recommended" (recomendado), significa que aún no se ha realizado ninguna calibración. Deberá calibrar el papel para garantizar los mejores resultados.
