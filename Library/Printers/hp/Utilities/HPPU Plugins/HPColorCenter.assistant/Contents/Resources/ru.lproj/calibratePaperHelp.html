Шаг 1. Выполните калибровку
Калибровка принтера требуется для выполнения повторного цвета для принтера, комбинации чернил и бумаги. Она осуществляется при помощи линеаризации принтера и определения максимально достижимой плотности для каждых чернил. Процедура включает фрагменты печати и измерений с различной плотностью отдельно для каждого типа чернил.

Шаг 2. Завершение
После завершения калибровки можно просмотреть список всех типов бумаги, данные о которых есть в принтере. Для каждого типа бумаги в списке приведена категория или специальное имя, состояние и дата калибровки. Если для состояния калибровки определенного типа отображается значение OK, это значит, что принтер недавно выполнял калибровку для данной бумаги и ее можно использовать. Если для состояния калибровки отображается значение Obsolete (Устарело), это значит, что калибровка уже выполнялась, но она больше недействительна, потому что прошло слишком много времени, печатающая головка была заменена или изменилось какое-то другое условие. Для достижения наилучших результатов нужно повторно выполнить калибровку принтера для этой бумаги. Если отображается значение Recommended (Рекомендуется), это значит, что калибровка ранее не выполнялась. Для достижения наилучших результатов нужно выполнить калибровку принтера для печати на этой бумаге.
