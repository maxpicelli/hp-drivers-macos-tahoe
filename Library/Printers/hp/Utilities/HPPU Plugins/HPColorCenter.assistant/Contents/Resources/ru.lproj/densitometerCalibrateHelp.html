Шаг 1. Выбор бумаги
Принтер должен периодически выполнять автоматическую калибровку для печати на выбранном типе бумаги. Калибровка выполняется для того, чтобы принтер работал стабильно и предсказуемо с каждым типом бумаги, данные о котором в нем имеются. Как только калибровка принтера завершена, он подготовлен для создания специального цветового профиля выбранной вами бумаги или для эффективного использования имеющегося цветового профиля. Всегда можно повторно калибровать принтер для использования такого же типа бумаги, чтобы вернуть его в стабильное рабочее состояние без необходимости создания нового цветового профиля. Для достижения наилучших результатов всегда можно повторно калибровать принтер или создать профиль, а также изменить заданный набор бумаги после того, как была выполнена первоначальная калибровка и создание профилей. Поступайте по своему усмотрению.

Шаг 2. Калибровка принтера
После выбора типа бумаги процесс калибровки включает следующие действия.
1. Печатается пробная диаграмма калибровки. Эта пробная диаграмма содержит пятна для получения которых израсходовано разное количество каждого вида чернил, из используемых в данном принтере; эти пятна часто называют также «чернильной шкалой», «эталонной шкалой» или «шкалой плотности». Пятна будут измерены (отсканированы) для определения двух основных параметров печати (наряду с другими параметрами): линейности и максимальной плотности каждой чернильной шкалы. Линейность означает, что каждое следующее пятно в серии (шкале) соответствует равному увеличению плотности (насыщенности). Максимальная плотность означает, какую плотность (насыщенность) можно получить для каждого цвета.
2. Отпечатанной пробной диаграмме нужно дать просохнуть в течение определенного времени. Диаграмме нужно обязательно дать высохнуть, прежде чем производить измерения, поскольку между влажным (сразу после печати) и сухим красителем (таким, как он будет выглядеть обычно) могут быть некоторые различия. Время просыхания зависит от типа бумаги.
3. С помощью встроенного денситометра принтера замеряется (сканируется) тестовая диаграмма. Денситометр — оптический инструмент для определения количества отраженного от цветного пятна света, которое в свою очередь соотносится с количеством чернил на бумаге.
4. Параметры калибровки вычисляются и сохраняются. На основе замеров, сделанных в пробной диаграмме, принтер вычисляет необходимые параметры корректировки, которые должны быть применены во время внутренней обработки изображения и сохранены для последующего использования. Впоследствии при печати на этом типе бумаги каждая чернильная шкала будет равномерной, и максимальная плотность будет соответствовать нужному уровню. Соблюдение этих условий является необходимым для успешного создания специального цветового профиля для выбранной бумаги или использования имеющейся.

Шаг 3. Завершение
После завершения калибровки можно просмотреть список всех типов бумаги, данные о которых есть в принтере. Для каждого типа бумаги в списке приведена категория или специальное имя, состояние и дата калибровки. Если для состояния калибровки определенного типа отображается значение OK, это значит, что принтер недавно выполнял калибровку для данной бумаги и ее можно использовать. Если для состояния калибровки отображается значение Obsolete (Устарело), это значит, что калибровка уже выполнялась, но она больше недействительна, потому что прошло слишком много времени, печатающая головка была заменена или изменилось какое-то другое условие. Для достижения наилучших результатов нужно повторно выполнить калибровку принтера для этой бумаги. Если отображается значение Recommended (Рекомендуется), это значит, что калибровка ранее не выполнялась. Для достижения наилучших результатов нужно выполнить калибровку принтера для печати на этой бумаге.
