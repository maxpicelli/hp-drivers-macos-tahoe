Etapa 1 – Selecionar o papel
Ocasionalmente, a impressora precisa ser calibrada para imprimir no seu papel. A calibragem é usada para colocar a impressora em um estado estável e repetitivo com cada tipo de papel que ela "reconhece". Depois de calibrada, a impressora estará pronta para desenvolver um perfil de cores personalizado para o seu papel ou usar o existente com bons resultados. Você poderá sempre recalibrá-la com o mesmo papel para restabelecer seu estado estável, sem ter de desenvolver outro perfil de cores. Para obter os melhores resultados, também é possível recalibrar e refazer o perfil para uma combinação entre impressora e papel a qualquer momento depois que a primeira calibragem e criação de perfis forem realizadas. Cabe a você decidir.

Etapa 2 – Calibrar a impressora
Depois de selecionar o tipo de papel a ser calibrado, o processo de calibragem consiste nas seguintes etapas:
1. Um gráfico do teste de calibragem é impresso contendo áreas com diversas porções de cada tinta usada na impressora, chamadas de "níveis de tinta", "níveis básicos" ou "níveis de densidade". As áreas serão medidas (digitalizadas) para determinar dois parâmetros básicos de impressão (entre outras coisas): linearidade e densidade máxima de cada nível de tinta. Linearidade é a correspondência de [a] cada área sucessiva na série (nível) a um aumento equivalente em densidade (escuridão). Densidade máxima é, simplesmente, quão densa (escura) cada tinta pode ficar ou está programada para ficar.
2. O gráfico de teste impresso leva um tempo predeterminado para secar. É importante permitir que o gráfico seque antes de ser medido, já que pequenas mudanças de cor podem ocorrer depois que a tinta úmida (logo após a impressão) seca (a forma como a impressão geralmente fica). O tempo de secagem usado depende do tipo de papel.
3. O gráfico de teste é medido (digitalizado) usando o densitômetro interno da impressora. Um densitômetro é um instrumento óptico que pode determinar a quantia de luz refletida de um patch de cores, que pode, por sua vez, estar relacionado à quantia de tinta no papel.
4. Os parâmetros de calibragem são calculados e armazenados. Com base nas medidas coletadas do gráfico teste, a impressora calcula os fatores de correção necessários para aplicar a seu canal interno de processamento de imagens e armazena-os para uso futuro. Na próxima vez que você imprimir nesse tipo de papel, cada nível de tinta será linear e a densidade máxima corresponderá à desejada. Essas condições básicas devem ser atendidas a fim de que a criação de um perfil de cores personalizado para o papel seja bem-sucedida ou para que se possa usar um perfil existente sem problemas.

Etapa 3 – Concluir
Quando a calibragem for concluída, você poderá percorrer a lista de todos os papéis que sua impressora "reconhece". Para cada papel, ela lista uma categoria ou um nome de papel personalizado, o status e a data de calibragem. Se o status de calibragem de um determinado papel for "OK", isso significa que a impressora foi calibrada para ele recentemente e está pronta para uso. Se o status for "Obsoleto", isso significa que a calibragem foi realizada pelo menos uma vez, mas não é mais confiável porque já se passou muito tempo, um cabeçote de impressão foi substituído ou alguma outra condição mudou. Você deve recalibrar a impressora para esse papel a fim de garantir os melhores resultados. Se o status for "Recomendado", isso significa que não foi realizada nenhuma calibragem ainda. Você deve calibrar a impressora para esse papel a fim de garantir os melhores resultados.
