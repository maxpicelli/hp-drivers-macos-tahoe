Paso 1 - Seleccionar papel
De vez en cuando, la impresora debe autocalibrarse para imprimir en el papel. La calibración se utiliza para ajustar la impresora en un estado estable y repetible con cada tipo de papel que "reconozca". Una vez que la impresora se haya calibrado, estará lista para crear un perfil de color personalizado para el papel, o utilizar el perfil de color ya existente con buenos resultados. Siempre podrá volver a calibrar la impresora con el mismo tipo de papel para reestablecer su estado estable, sin necesidad de tener que crear un nuevo perfil de color para dicho tipo de papel. Para obtener los mejores resultados, también puede optar por volver a realizar la calibración y volver a crear un perfil para la combinación de la impresora y un determinado tipo de papel en cualquier momento, después de haberse realizado la calibración inicial y haberse creado un perfil. Usted elige lo que más le convenga.

Paso 2 - Calibrar la impresora
Una vez que haya seleccionado el tipo de papel que se va a calibrar, el proceso de calibración consiste en los siguientes pasos:
1. Se imprime un gráfico de prueba de calibración. El gráfico de prueba contiene parches con diversas cantidades de cada una de las tintas que se utilizan en la impresora, a menudo referidos como "rampa de tinta", "rampa primaria" o "rampa de densidad". Los parches serán medidos (escaneados) para determinar dos parámetros de impresión básicos (entre otras cosas): linealidad y máxima densidad de cada rampa de tinta. Linealidad significa que cada parche sucesivo de la serie (rampa) corresponde a un incremento equitativo de densidad (oscuridad). Máxima densidad significa simplemente el grado de densidad (oscuridad) que puede alcanzar cada tinta, o que se le permite alcanzar.
2. El gráfico de prueba impreso puede dejarse secar durante un tiempo preestablecido. Es importante dejar que el gráfico se seque antes de medirlo, ya que pueden producirse pequeñas variaciones de color entre la tinta que aún está húmeda (justo después de imprimirse) y la tinta seca (el aspecto normal que tendrá la impresión). El tiempo de secado empleado depende del tipo (o categoría) del papel.
3. El gráfico de prueba se mide (escanea) a través del espectrofotómetro integrado en la impresora. Un espectrofotómetro es un instrumento de precisión que puede determinar la composición exacta de la luz que refleja un parche de color. Divide la luz reflejada en componentes con diferente longitud de onda, del mismo modo que el famoso prisma de Newton dividía la luz blanca en los colores del arco iris, y mide la fuerza de cada componente.
4. Los parámetros de la calibración se calculan y se almacenan. Basándose en las mediciones obtenidas a partir del gráfico de prueba, la impresora calcula los factores de corrección que necesita aplicar a su procesamiento de imágenes interno y los almacena para un futuro uso. La siguiente vez que imprima en este tipo de papel, cada rampa de tinta será lineal y la máxima densidad se corresponderá con la deseada. Éstas son las condiciones básicas que deberán cumplirse para la correcta creación de un perfil de color personalizado para el papel o para el uso de un perfil ya existente.

Paso 3 - Finalizado
Una vez finalizada la calibración, puede consultar la lista de todos los tipos de papel que reconoce su impresora. Para cada tipo de papel se indica: la categoría de papel, el nombre de papel, el estado de calibración y la fecha de calibración. Si en el estado de calibración de un determinado papel aparece "OK" (correcto), significa que dicho papel ha sido calibrado recientemente y está listo para ser utilizado. Si el estado es "Obsolete" (obsoleto), significa que se ha realizado una calibración al menos una vez, pero se considera que ha transcurrido demasiado tiempo desde entonces para que ésta resulte fiable, o ya no se considera válida, por ejemplo, debido al cambio de un cabezal de impresión. Deberá volver a calibrar el papel para garantizar los mejores resultados. Si el estado es "Recommended" (recomendado), significa que aún no se ha realizado ninguna calibración. Deberá calibrar el papel para garantizar los mejores resultados.
