Der Bereich "Telefonbucheinstellungen" dient zum Hinzufügen von Gruppen und/oder Kontakten zum Telefonbuch des Geräts.

HINWEIS: Von einigen Geräten mit Telefonbuch werden keine Gruppen unterstützt. Weitere Informationen finden Sie in der Dokumentation zu Ihrem Gerät.

Klicken Sie zum Erstellen eines neuen Einzeleintrags auf die Schaltfläche +, oder doppelklicken Sie auf ein Feld, und beginnen Sie mit der Eingabe. Sobald Sie mit der Eingabe von Einträgen aus Ihrem Adressbuch beginnen, wird ein Dialogfenster angezeigt, in dem das Feld Name automatisch ausgefüllt wurde. Das Feld Faxnummerwird automatisch dem ausgewählten Eintrag entsprechend ausgefüllt.

Sie können auch auf das Symbol HSDPhoneBookIconPlaceholder klicken, um Einträge aus dem Adressbuch hinzuzufügen. Doppelklicken Sie auf einen Eintrag, oder wählen Sie einen Eintrag aus und klicken Sie auf Hinzufügen. Der Eintrag wird im ersten nicht zugewiesenen Telefonbuchdatensatz hinzugefügt. Sie können Einträge auch auf Positionen ohne zugeordnete Nummer ziehen.

Wählen Sie zum Erstellen einer neuen Gruppe die Einträge für die Gruppe aus, und klicken Sie auf HSDOptionsIconPlaceholder. Wählen Sie dann im Menü die Option Gruppe erstellen aus. Wählen Sie zum Bearbeiten einer Gruppe die Gruppe und dann im Menü die Option Bearbeiten aus.

Wählen Sie zum Löschen einer Gruppe oder eines Einzeleintrags, die Gruppe bzw. den Eintrag aus, und klicken Sie auf die Schaltfläche -.

Klicken Sie auf "Liste drucken", um alle auf dem Gerät gespeicherten Telefonbuchdatensätze zu drucken.

HINWEIS: Sie können die Shift oder die Command gedrückt halten und klicken, um eine Mehrfachauswahl zu treffen.
