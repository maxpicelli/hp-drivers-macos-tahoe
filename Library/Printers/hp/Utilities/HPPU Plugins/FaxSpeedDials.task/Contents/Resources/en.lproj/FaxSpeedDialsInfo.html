The Speed Dial settings pane is used to enter groups and/or individuals into the Speed Dial memory on your device.

NOTE: Not all devices with Speed Dial support Groups. For more information, see the documentation that came with your device.

To create a new single entry, click the + button or double-click in a blank field and start typing. A dialog will appear in which the Name field will auto-fill as you type using entries from your Address Book. The Fax Number field will be automatically populated with the selected entry.

You can also click the HSDPhoneBookIconPlaceholder icon to add entries from your Address Book. Double click an entry or select the entry and click the Add button. The entry will be added to the first unassigned Speed Dial location. You can also drag entries into unassigned locations.

To create a new group, select the entries for the group and click HSDOptionsIconPlaceholder and choose Create Group from the menu. To edit a group, select the group and then choose Edit from the menu.

To delete a group or individual entry, select it and click the - button.

Click “Print List” to print a report of your current Speed Dial settings.

Hint: You can Shift-click and Command-click to make multiple selections.
