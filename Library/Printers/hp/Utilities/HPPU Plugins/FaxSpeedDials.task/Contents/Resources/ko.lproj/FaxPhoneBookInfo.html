전화번호부 설정 패널을 사용하여 장치의 전화번호부에 그룹 및/또는 연락처를 추가할 수 있습니다.

참고: 일부 장치는 전화번호부가 있어도 그룹을 지원하지 않습니다. 자세한 정보를 확인하려면 장치와 동봉된 설명서를 참조하십시오.

새 항목을 만들려면 + 단추를 누르거나 빈 필드에 두 번 클릭하고 입력합니다. 표시되는 대화 상자에 주소록 항목을 입력하면 이름 필드에 항목들이 자동 입력됩니다. 팩스 번호 필드에 선택한 항목이 자동으로 나타납니다.

HSDPhoneBookIconPlaceholder 아이콘을 눌러서 주소록에서 항목을 추가할 수 있습니다. 항목을 두 번 클릭하거나 선택한 후 추가 단추를 누릅니다. 항목이 할당되지 않은 첫 번째 전화번호부 레코드에 추가됩니다. 또는 지정 안 된 위치에 항목을 끌어올 수 있습니다.

새 그룹을 만들려면 항목을 선택한 후 HSDOptionsIconPlaceholder를 누르고 메뉴에서 그룹 만들기를 선택합니다. 그룹을 편집하려면 선택한 후 메뉴에서 편집을 누릅니다.

그룹 또는 개별 항목을 제거하려면 선택한 후 – 단추를 누릅니다.

"목록 인쇄"를 클릭하여 이 장치에 저장된 모든 전화번호부 레코드를 인쇄합니다.

힌트: 다중 선택하려면 Shift 클릭 또는 Command 클릭합니다.
