Le volet des paramètres de numéro abrégé permet d’entrer des groupes et/ou des individus dans la mémoire de numéros abrégés de votre appareil.

REMARQUE : Les appareils équipés de la fonction Numéro abrégé ne prennent pas tous en charge les groupes. Pour plus d’informations, veuillez vous reporter à la documentation fournie avec votre appareil.

Pour créer une nouvelle entrée individuelle, cliquez sur le bouton + ou double-cliquez sur un champ vide et commencez à saisir les informations. Une boîte de dialogue apparaît et le champ Nom se remplit automatiquement à mesure de votre saisie en utilisant les entrées de votre répertoire. Le champ N° de télécopie se remplit automatiquement avec l’entrée sélectionnée.

Vous pouvez également cliquer sur l’icône HSDPhoneBookIconPlaceholder pour ajouter des entrées à partir de votre répertoire. Double-cliquez sur une entrée ou sélectionnez l’entrée et cliquez sur le bouton Ajouter. L'entrée sera ajoutée au premier emplacement de numéro abrégé non attribué. Vous pouvez également faire glisser des entrées vers des emplacements non attribués.

Pour créer un nouveau groupe, sélectionnez les entrées pour le groupe et cliquez sur HSDOptionsIconPlaceholder puis choisissez Créer un groupe dans le menu. Pour modifier un groupe, sélectionnez le groupe, puis choisissez Modifier dans le menu.

Pour supprimer un groupe ou une entrée individuelle, sélectionnez l’élément et cliquez sur le bouton -.

Cliquez sur “Imprimer la liste” pour imprimer un rapport sur vos paramètres de numéros abrégés actuels.

Conseil : vous pouvez utiliser les combinaisons Shift+clic ou Command+clic pour effectuer des sélections multiples.
