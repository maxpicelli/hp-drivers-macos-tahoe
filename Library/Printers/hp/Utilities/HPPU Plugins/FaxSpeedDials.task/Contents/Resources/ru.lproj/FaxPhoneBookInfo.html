Панель настроек телефонной книги используется для добавления групп или контактов в телефонную книгу устройства.

ПРИМЕЧАНИЕ. Некоторые из устройств с телефонной книгой не поддерживают группы. Дополнительную информацию см. в документации, прилагаемой к устройству.

Для создания отдельной записи щелкните кнопку «+» или дважды щелкните в пустом поле и начните ввод. Появится диалоговое окно, в котором поле Имя будет автоматически заполняться по мере ввода с помощью записей из адресной книги. Поле Номер факса будет автоматически заполняться с помощью выбранной записи.

Можно также щелкнуть значок HSDPhoneBookIconPlaceholder для добавления записей из адресной книги. Дважды щелкните запись или выберите запись и нажмите кнопку Добавить. Запись будет добавлена в первую свободную ячейку телефонной книги. Можно также перетаскивать записи в свободные места.

Чтобы создать новую группу, выберите записи для группы и щелкните HSDOptionsIconPlaceholder, а затем выберите Создать группу в меню. Для изменения группы выберите ее, а затем выберите Изменить в меню.

Чтобы удалить группу или отдельную запись, выберите ее и нажмите кнопку «-».

Щелкните на "Печать списка" для печати всех записей телефонной книги на этом устройстве.

Совет. Чтобы выбрать мышью сразу несколько элементов, удерживайте при этом нажатой клавишу Shift или Command.
