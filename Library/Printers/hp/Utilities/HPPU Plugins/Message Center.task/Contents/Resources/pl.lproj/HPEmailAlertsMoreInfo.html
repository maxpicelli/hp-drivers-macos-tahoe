Do odbiorców zostanie wysłana wiadomość e-mail z ostrzeżeniem, jeśli materiał eksploatacyjny osiągnie któryś z następujących stanów:
  • Niski poziom toneru
  • Bardzo niski poziom toneru/brak toneru

Domyślnym klientem poczty e-mail powinna być jedna z poniższych aplikacji:
  • Mail
  • Microsoft Entourage
  • Microsoft Outlook
