Предупреждение по эл. почте будет отправлено получателям, если расходные материалы находятся в указанном ниже состоянии:
  • осталось небольшое количество тонера;
  • осталось очень небольшое количество тонера/тонер закончился;

Клиентом эл. почты по умолчанию должно стать одно из следующих приложений:
  • Mail
  • Microsoft Entourage
  • Microsoft Outlook
