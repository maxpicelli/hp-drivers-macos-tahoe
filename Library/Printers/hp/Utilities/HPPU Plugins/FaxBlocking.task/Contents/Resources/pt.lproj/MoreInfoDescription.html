O painel de configurações de Bloqueio de fax indesejado é usado para inserir uma lista de números de fax que você deseja impedir que enviem faxes para o seu dispositivo.

O Bloqueio de fax indesejado requer que o seu dispositivo esteja conectado a uma linha com identificação de chamadas ativa.

Se você não sabe se a sua linha tem a identificação de chamadas ativa, entre em contato com o seu provedor de serviços de telefonia.

Clique em “Imprimir lista” para imprimir um relatório dos números bloqueados.

DICA: Você pode pressionar os botões Shift ou Command, quando clicar, para escolher vários itens.
